# Usage

Add in your ESLint config

```json
{
	"extends": [
		"mahir/common",
		"mahir/node",
		"mahir/module",
		"mahir/typescript",
		"mahir/react",
		"mahir/next",
		"mahir/edge",
		"mahir/prettier"
	]
}
```

You can remove any of the configs you don't need.
